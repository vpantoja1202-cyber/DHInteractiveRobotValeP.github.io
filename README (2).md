# Denavit-Hartenberg Interactivo

Página web interactiva sobre el método **Denavit-Hartenberg** para la representación y cálculo de cinemática en robots industriales.

## ✨ Contenido

- **Teoría** — Explicación de los 4 parámetros D-H (θ, d, a, α)
- **Tabla D-H** — Tabla comentada para distintos robots (KUKA, SCARA, Cartesiano, Planar, Anthropomorphic)
- **Simulador 3D** — Visualización 3D interactiva con Three.js, control de ángulos articulares en tiempo real
- **Cinemática Inversa** — Módulo de IK con visualización 3D del brazo robótico
